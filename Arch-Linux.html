Uso basico de Arch Linux


Uso de Pacman (gestor de paquetes)

pacman -S Instala el pacquete que indiques

pacman -Ss Busca un paquete en los repositorios

pacman -Syu Actualizacion del sistema

pacman -Syu Actualizar un paquete

pacman -Syy Actualiza los repositorios

pacman -R Elimina un paquete pero no las dependencias

pacman -Rs Elimina el paquete y las dependencias


Activar el repositorio Multilib

1. Editar /etc/pacman.conf

2. Descomentar las lineas:

[multilib]

Include = /etc/pacman.d/mirrorlist


Habilitar AUR (repositorio de la comunidad)

1. Instalar git

2. clonar el repositorio oficial de yay

git clone https://aur.archlinux.org/yay.git

3. ir a la ruta en la que se a instalado yay

cd yay

4. instalar el paquete

makepkg -si

sudo pacman -S --needed git base-devel && git clone https://aur.archlinux.org/yay.git && cd yay && makepkg -si
